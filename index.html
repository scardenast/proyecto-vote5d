Privacidad del Votante

Se utilizan técnicas de encriptación avanzada y disociación de datos para asegurar que el voto sea anónimo y no pueda ser rastreado hasta el individuo. Los datos biométricos se procesan localmente y se utilizan solo para la autenticación.

Detección de Anomalías con Machine Learning

Algoritmos de IA monitorean patrones de votación en tiempo real para detectar y alertar sobre comportamientos inusuales o posibles intentos de fraude, sin comprometer la privacidad del voto.

Trazabilidad con Blockchain

Cada voto emitido (de forma anónima) se registra en una cadena de bloques distribuida, garantizando su inmutabilidad y permitiendo auditorías transparentes y verificables por cualquier entidad autorizada.

Aplicaciones de Inteligencia Artificial

Validación Facial Adaptativa

Modelos de IA que se adaptan a cambios en la apariencia y condiciones de iluminación.

Reconocimiento de Voz Seguro

Sistemas que analizan múltiples características de la voz para una autenticación precisa.

Firma Digital Dinámica

Análisis de la biometría del trazo de la firma para una capa adicional de seguridad.

Análisis de Sentimiento Ciudadano

Procesamiento de lenguaje natural (NLP) para analizar feedback y opiniones sobre el sistema.

Clustering Predictivo de Ubicaciones

IA para identificar las ubicaciones óptimas para las cabinas, maximizando la accesibilidad.

Optimización de Procesos

Uso de IA para optimizar la logística electoral y la asignación de recursos.

Preguntas Frecuentes y Ética del Sistema

El sistema VOTE-5D está diseñado con redundancia. Si un factor biométrico falla, el sistema permite usar combinaciones alternativas. Como medida de contingencia excepcional, se podría recurrir a una validación manual asistida.

Los datos biométricos se procesan localmente (edge computing) y se convierten en plantillas encriptadas (hashes) para autenticación, luego se descartan o almacenan disociados, cumpliendo Ley N° 19.628. El voto es anónimo.

El acceso es estrictamente controlado. SERVEL y organismos autorizados acceden a resultados agregados y datos de auditoría. Los votos son anónimos y las plantillas biométricas no se vinculan al voto.

Múltiples niveles: código fuente, seguridad (pentesting), infraestructura, procesos, y auditoría ciudadana (blockchain). Auditorías post-electorales por organismos competentes.
